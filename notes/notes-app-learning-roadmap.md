# Notes App React Learning Roadmap

## Level 0 --- Static UI

Membangun struktur component tanpa data dinamis.

## Level 1 --- NotesForm State

NotesForm mampu:

-   menyimpan title
-   menyimpan body
-   menggunakan controlled input
-   menghitung derived data

## Level 2 --- NotesCard Props

NotesCard mampu menerima data melalui props.

## Level 3 --- Notes List Rendering

Mengubah data array menjadi component menggunakan map dan key.

## Level 4 --- State Ownership

NotesMain menjadi pemilik notes state dan menggunakan callback props.

## Level 5 --- CRUD Notes

Create, delete, archive, restore menggunakan immutable update.

## Level 6 --- Search & Derived Data

Menggunakan state dan derived data untuk filtering.

## Level 7 --- Persistence

Menambahkan localStorage atau API.

## Level 8 --- Production Pattern

Mempelajari custom hooks, useReducer, Context, testing, dan performance.
